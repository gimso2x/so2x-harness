# Spec: v0.3 Knowledge Accumulation System

## Goal
프로젝트 간 학습을 축적, 검색, 공유하는 시스템을 추가한다.
JSONL 기반 저장, so2x-cli로 관리, /specify에서 자동 연동.

## Key Decisions
1. JSONL 파일로 학습 저장 (git 추적 가능)
2. 프로젝트 로컬 + 중앙(so2x-harness repo) 이중 저장
3. 키워드 기반 검색 (정규식, 태그 필터)
4. /specify 실행 시 관련 학습 자동 주입
5. so2x-cli learn 명령으로 CRUD

## Requirements
- R1: 학습 엔트리 JSONL 형식 (id, timestamp, source, category, problem, cause, rule, tags, severity)
- R2: so2x-cli learn add/search/sync/summary 명령
- R3: 프로젝트 로컬 learnings.jsonl + 중앙 learnings/entries.jsonl
- R4: /specify에서 키워드 기반 자동 학습 검색
- R5: 카테고리: pattern, anti-pattern, edge-case, decision

## Verification
- so2x-cli learn add 가 정상 동작
- so2x-cli learn search 가 키워드로 학습 검색
- so2x-cli learn sync 가 프로젝트→중앙 동기화
- make test 통과
